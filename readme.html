2. w4_git clone

sshでremote loginが可能になるように設定します． これは，system連携をhostをまたいで可能にすることを意味します． 

system "ssh-keygen" # return only!! no PassWord!!
system "cat ~/.ssh/id_ed25519.pub" # then Copy&Paste
system "explorer.exe https://github.com/daddygongon" # ssh key
# and copy multi_scale_25f ssh path

system "git clone git@github.com:daddygongon/multi_scale_25f.git"
